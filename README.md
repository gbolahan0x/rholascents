# RholaScents 🛍️

> Nigeria's premier destination for authentic luxury fragrances.

**Live Site:** https://Gbolahan0x.github.io/rholascents

---

## Features
- 🔥 Bold product catalog with category filters
- 🛒 Shopping bag with quantity controls
- 💳 OPay checkout integration
- 💬 WhatsApp ordering (auto-formatted cart message)
- 📢 Animated promo banners & ad ticker
- 📱 Fully responsive (mobile & desktop)

## Setup
1. Clone the repo
2. Update WhatsApp number in `index.html` (`+2348000000000`)
3. Add your OPay merchant checkout URL
4. Push changes — site auto-updates via GitHub Pages

## Stack
- Pure HTML / CSS / JavaScript (zero dependencies)
- Single-file build — fast, portable, easy to maintain
